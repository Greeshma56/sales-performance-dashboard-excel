Sales Performance Dashboard using Microsoft Excel

Project Overview
Developed an interactive Sales Performance Dashboard using Microsoft Excel to analyze 500+ sales transactions.

Features
- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Average Order Value KPI
- Region-wise Sales Analysis
- Monthly Sales Trend Analysis
- Top Selling Products Analysis
- Interactive Slicers

 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Excel Formulas
